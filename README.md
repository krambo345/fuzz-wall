# fzwall
A fuzzy wallpaper picker using fuzzel

## Dependencies
- fuzzel
- one of: swaybg, swww, feh, nitrogen, xwallpaper

## Install
### AUR
paru -S fzwall
yay -S fzwall

## Usage
fzwall
# or set custom wallpaper dir:
FUZZEL_WALLPAPER_DIR=~/Pictures/walls fzwall

## Supported WMs
- Sway, Hyprland, river (Wayland)
- i3, bspwm, openbox (X11)
