# Security Policy

## Supported Versions
| Version | Supported |
|---------|-----------|
| 1.x     | ✅        |

## Reporting a Vulnerability
Please **do not** open a public issue for security vulnerabilities.

Email: contact@aditya-verma.me

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

You'll get a response within 48 hours.
